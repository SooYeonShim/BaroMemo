# ✦ BaroMemo (바로메모)

> **"생각나는 즉시, 기록은 바르게."**  
> 복잡한 설정 없이 실행 파일 하나로 즐기는 미니멀하고 강력한 데스크톱 메모 앱입니다.

![BaroMemo Main UI](https://via.placeholder.com/800x450.png?text=BaroMemo+Interface+Preview)

---

## ✨ 핵심 기능 (Key Features)

### 1. 🚀 설치가 필요 없는 포터블 앱
*   빌드 후 생성되는 단일 실행 파일(`.exe`)만 있으면 어디서든 바로 실행 가능합니다.
*   가벼운 리소스 점유율로 저사양 PC에서도 쾌적하게 동작합니다.

### 2. 📝 직관적인 에디터 경험
*   **즉시 서식 적용**: 새 메모 작성 시에도 단축키(`Ctrl+B`, `Ctrl+I` 등)로 서식을 먼저 정하고 바로 타이핑을 시작할 수 있습니다.
*   **깔끔한 텍스트 관리**: 제목과 본문을 분리하여 관리하며, 제목에는 불필요한 서식이 섞이지 않도록 보호합니다.
*   **다양한 도구**: 글머리 기호, 번호 목록, 이미지 삽입 및 폰트 크기 조절을 지원합니다.

### 3. 📂 체계적인 카테고리 & 태그
*   사용자 정의 카테고리로 메모를 분류하고 드래그 앤 드롭으로 간편하게 이동할 수 있습니다.
*   해시태그 시스템을 통해 카테고리 전반에 걸친 빠른 필터링을 지원합니다.

### 4. 🗑️ 스마트한 휴지통 관리
*   삭제된 메모는 휴지통에서 안전하게 보관됩니다.
*   **문맥 유지**: 휴지통을 비운 후에는 이전에 작업하던 카테고리로 자동 복귀하여 작업의 연속성을 보장합니다.

*   **Frontend**: HTML5, CSS3, Vanilla JavaScript
*   **Runtime**: [Electron](https://www.electronjs.org/)
*   **Packaging**: [Electron-Builder](https://www.electron.build/)

## 🚀 시작하기 (Getting Started)

### 사전 준비
*   [Node.js](https://nodejs.org) 설치가 필요합니다.

### 빌드 및 실행
1. 저장소를 클론하거나 다운로드합니다.
2. `build.bat` 파일을 실행합니다.
3. 생성된 `dist\BaroMemo.exe` 파일을 실행하여 바로 사용하세요!
